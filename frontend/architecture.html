FastAPI Reference

Active endpoints served by the backend loop on http://127.0.0.1:8000

GET /smart-grid-status

Returns the latest computed state of the grid. Powered by a background asyncio loop running every 10 seconds.

{ "weather": { "temperature": 38.0, "cloud_cover": 42, ... }, "solar_generation": 72.5, "total_demand": 80.0, "battery_before": 50.0, "backup_used": 7.5, "battery_after": 42.5, "optimized_distribution": { "Hospital": { "demand": 15, "allocated": 15, "status": "fully_met" }, ... } }
GET /battery-history

Returns historical battery state dumps from the internal CSV log.

GET /forecast-tomorrow

Predicts tomorrow's energy balance based on the current weather parameters offset by 5°C module temperature difference.
