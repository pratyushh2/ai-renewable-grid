Running the System Locally

To run the digital twin yourself, follow these instructions:

  1. Ensure you have python 3.x installed.
  2. Install dependencies from the root directory:

    pip install -r requirements.txt

  3. Run the FastAPI backend server (from root):

    uvicorn backend.app:app --reload

  4. Serve the frontend. You can use Python's built-in http server:

    cd frontend && python -m http.server 3000

  5. Navigate to http://localhost:3000 in your browser.

AI Assistant (GridAI)

The GridAI Copilot is a custom rule-based pattern matching agent directly injected into the frontend Javascript. It avoids high-latency external API calls such as OpenAI, allowing operators to get instant natural-language breakdown of grid conditions without the overhead.
